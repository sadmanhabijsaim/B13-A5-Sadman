## 📝 Conceptual Questions (Self-Answered)

### 1️⃣ What is the difference between var, let, and const?
- **var:** Eta purono version-er JavaScript-e bebohar kora hoto. Er main problem holo eta function-scoped, mane block-er baireo access kora jay jeta onno code-e impact fele.
- **let:** Eta modern way. Eta block-scoped, mane shudhu nirdishto curly braces `{}` er bhitor kaj kore. Er value change kora jay.
- **const:** Etau block-scoped, kintu er value ekbar assign korle ar change kora jay na.

### 2️⃣ What is the spread operator (...)?
- Spread operator (...) diye ekta array ba object-er shob components-ke khule onno jaygay copy ba merge kora jay. Jemon: `[...oldArray, newItem]`. Eta original data-ke change na kore copy korte sahajjo kore.

### 3️⃣ What is the difference between map(), filter(), and forEach()?
- **map():** Eta array-er prottek upadan niye kaj kore ebong result hishebe ekta "notun array" return kore.
- **filter():** Eta shudhu oi elements gulo diye notun array banay jara kono nirdishto condition (shortho) puron kore.
- **forEach():** Eta array-er prottekta item-er upore loop chalay kintu kono kichu return kore na.

### 4️⃣ What is an arrow function?
- Arrow function holo JavaScript-e function lekhar ekta short ebong clean method. Ekhane `function` keyword-er bodole `=>` (fat arrow) use kora hoy. Eta code dekhte beshi readable kore.

### 5️⃣ What are template literals?
- Template literals holo backtick (`` ` ``) use kore string lekhar paddhati. Er boro subidha holo `${variable}` use kore string-er bhitorei dynamic variable boshano jay, jeta normal single ba double quote-e kora jay na.
